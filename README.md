## Random Quote
This is an Random Quote generator application built with ReactJS. Every on browser reload or new button press you will get a new quote. The quote are generated from API.

## Features
- Random quote
- Author name
- Background color
- Share on Twitter & Facebook
- New quote button
- Fully responsive
